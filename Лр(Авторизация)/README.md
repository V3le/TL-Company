# Лабораторная работа - Система авторизации

## Описание
Автономная система авторизации и регистрации пользователей с использованием существующей базы данных `tl_web`. Полностью независимый проект со всеми необходимыми файлами.

## Структура проекта
```
Лр(Авторизация)/
├── index.html              # Главная страница
├── images/
│   └── logo.png           # Логотип
├── css/
│   ├── fonts.css          # Подключение шрифтов
│   ├── design-system.css  # Базовые стили
│   └── auth.css           # Стили авторизации
├── js/
│   └── auth.js            # Логика авторизации
└── api/
    ├── config/
    │   └── database.php   # Подключение к БД
    ├── models/
    │   └── User.php       # Модель пользователя
    └── auth/
        ├── login.php           # Вход
        ├── register.php        # Регистрация
        ├── logout.php          # Выход
        ├── check-session.php   # Проверка сессии
        ├── check-email.php     # Проверка email
        └── check-username.php  # Проверка логина
```

## Функционал

### Авторизация
- Вход по логину или email
- Опция "Запомнить меня"
- Валидация полей
- Показ/скрытие пароля

### Регистрация (2 шага)
**Шаг 1:**
- Логин (проверка уникальности)
- Email (проверка уникальности и формата)
- Пароль (индикатор надёжности)
- Подтверждение пароля

**Шаг 2:**
- Фамилия, Имя, Отчество
- Дата рождения
- Телефон (маска +7)
- Город
- Согласие с политикой

### Профиль пользователя
- Отображение инициалов
- Информация о пользователе
- Кнопка выхода

## Запуск

1. Убедитесь, что у вас запущен локальный сервер (Apache + MySQL)
2. База данных `tl_web` должна быть создана и содержать таблицу `users`
3. Откройте в браузере: `http://localhost/Лр(Авторизация)/`
4. Или запустите `start.bat` для автоматического открытия в браузере

## Важно
✅ Проект полностью автономен и не зависит от других папок
✅ Все необходимые файлы (логотип, стили, скрипты) находятся внутри папки `Лр(Авторизация)`
✅ Использует существующую БД `tl_web` и таблицу `users`

## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+, Fetch API)
- PHP 7.4+
- MySQL
- PDO для работы с БД

## Особенности
- Адаптивный дизайн
- Валидация в реальном времени
- Toast-уведомления
- Безопасное хранение паролей (bcrypt)
- Управление сессиями
- Красивые анимации и переходы
- Социальные кнопки с анимацией

## База данных
Использует существующую таблицу `users` из БД `tl_web` со следующими полями:
- id
- username
- email
- password
- first_name
- last_name
- middle_name
- birth_date
- phone
- city
- is_active
- created_at
